Vector

头文件

定义

构造函数

属性

vector可以理解成一个可以扩张的数组，每次扩张多出来一定数量的容量。这个多出多少是不确定的，不同编译器不同，VS2015是1.5，GCC是2倍，一般介于这两者之间。

  1. size_type capacity()
  • 返回当前容量
  • 容量并非实际有几个元素，而是最多能放几个元素
  1. void reserve( size_type size )
  • 设置容器可以容纳几个元素
  1. size_type size()
  • 返回当前元素数量
  1. void resize( size_type size, TYPE val )
  • 改变容器元素数量
  • 会强制改变元素数量无论减少还是增加，都填为val
  1. bool empty()
  • 判断vector是否为空（返回true时为空）
  1. void clear()
  • 清空所有元素

操作

  1. void push_back( const TYPE &val )
  • 在容器末尾添加一个元素
  1. iterator insert( iterator loc, const TYPE &val )
    void insert( iterator loc, size_type num, const TYPE &val )
    void insert( iterator loc, input_iterator start, input_iterator end )
  • 插入元素
  1. void pop_back()
  • 移除最后一个元素
  1. iterator erase( iterator loc )
    iterator erase( iterator start, iterator end )
  • 删除指定元素
